# Story S3-04 — TUI connection indicator

## Description
Afficher l'état de connexion à opencode dans la zone TUI.

## Acceptance Criteria
- [ ] Indicator visible en header de la TUI zone
- [ ] Vert + "Connecté" quand opencode répond
- [ ] Rouge + "Déconnecté" quand erreur de connexion
- [ ] Health check toutes les 10s via `/api/health`
- [ ] Tooltip avec URL d'opencode

## Effort
S (Small)

## Dependencies
Aucune

## Files to modify
- `public/index.html` — indicator dans TUI header
- `public/app.js` — fonction `checkOpencodeConnection()` + UI update
