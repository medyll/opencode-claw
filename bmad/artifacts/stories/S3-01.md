# Story S3-01 — Zone info header projet

## Description
Afficher les informations principales du projet sélectionné dans la zone info droite.

## Acceptance Criteria
- [ ] Header affiche: nom du projet, path absolu, statut (active/running/error)
- [ ] Badge de statut coloré (vert=active, orange=running, rouge=error)
- [ ] Path tronqué avec tooltip si trop long
- [ ] Mis à jour lors du changement de projet
- [ ] Design cohérent avec css-base

## Effort
S (Small)

## Dependencies
Aucune

## Files to modify
- `public/index.html` — ajouter le header dans la zone info
- `public/app.js` — fonction `renderProjectInfo(project)`
